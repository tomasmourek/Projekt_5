# Dostupnost potravin v kontextu vývoje mezd

Power BI projekt zaměřený na analýzu vývoje cen vybraných potravin a hrubých mezd v České republice. Cílem je ukázat, jak se v čase měnila dostupnost potravin v závislosti na vývoji příjmů v jednotlivých odvětvích.

Report pracuje s časovými řadami a umožňuje interaktivní filtrování podle roku, komodity a typu zaměstnání. Součástí je také porovnání potravin a výpočet, kolik jednotek dané komodity lze koupit za průměrnou mzdu.

Data použitá v projektu pocházejí z mého předchozího projektu Engeto.
